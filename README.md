## 🧠 About The Project

A **responsive, animated single-page portfolio** built with **React.js** to professionally showcase my skills, projects, education, certifications, and contact details. Designed with smooth transitions and clean, interactive UI to reflect my personal and academic journey in tech.

> 🎯 Built for recruiters, collaborators, and anyone curious about my work — clean, fast, and fully mobile-responsive.

## 🚀 Getting Started

**1. Clone the repository**

```bash
git clone https://github.com/aryansengar007/my-portfolio-v1.git
cd my-portfolio-v1
```

**2. Install dependencies**

```bash
npm install
```

**3. Start the development server**

```bash
npm start
```

> ✅ Opens at `http://localhost:3000` with hot reload enabled.

## 🌍 Deployment

The site is deployed on **Netlify**. To deploy your own version:

**1. Build the production bundle**

```bash
npm run build
```

**2. Deploy to Netlify**

Drag and drop the generated `/build` folder into your [Netlify dashboard](https://app.netlify.com/).

**3. Fix client-side routing**

Add a `_redirects` file inside the `public/` folder with the following content:

```
/*    /index.html    200
```

> This ensures React Router works correctly on Netlify for all routes.
